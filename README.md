# FlashOfP　ソースコード
タイトル：Pの閃き<br>
コンセプト：少しの閃きとアクション<br>
ターゲット：閃きクイズが好きな中学生<br>
開発環境：Unity, C#, Microsoft Visual Studio<br>
全5ステージ<br>
ゲームジャンル：2Dのアクション×パズルゲーム<br>

ゲーム概要：pが欠けている英単語を完成させ、その効果を用いてゴールを目指す2Dのアクション×パズルゲームです<br>
プレイヤーが操作するpの文字は移動しかできません。しかし、「jum」や「dro」といった文字がステージ内にあり<br>
そこにpを当てはめると様々な能力が使えるようになります<br>
頭の中で、この文字を作ればこうなりそうと考えながら、ゴールへ導くゲームとなっています<br>
このゲームは閃きも大事ですが、アクションを大事にしています。<br>
解法が分かったとしても、ゴールにたどり着くにはアクションも必要。そんなゲームとなっております。<br>

### こだわった点

・レベルデザインで少しずつ難しくなっていくステージ構成<br>
・ゴール演出や言葉を完成させるときの遊び心のあるアニメーション<br>
・言葉によるギミックとp使って文字を完成させる世界観<br>
・普通のクリアと、条件付きクリアを用意したことにより、2度楽しめるような設計を用意<br>
